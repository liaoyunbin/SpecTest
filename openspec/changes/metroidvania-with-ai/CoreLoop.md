# CoreLoop — 类银河城游戏核心循环

## 30 秒核心循环

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   探索未知区域 → 遭遇敌人/障碍 → 战斗/破解 → 获得奖励           │
│       ↑                                            │             │
│       │                                            ▼             │
│       └────────── 解锁新能力 ──── 击败Boss/找到关键道具          │
│                                            │                    │
│                                            ▼                    │
│                              新能力 → 打开之前无法通过的门/区域   │
│                                            │                    │
│                                            ▼                    │
│                              回到之前的地图 → 发现新路径/隐藏区域 │
│                                                                 │
│   "探索 → 受阻 → 成长 → 回溯 → 再探索" 的无限正向循环            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**一句话概括：** 玩家在一个庞大相连的世界中探索，获取新能力解锁新区域，在不断的"遇到障碍 → 变强 → 克服"的循环中体验成长感与发现的乐趣。

## 游戏三大支柱

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│         支柱 1                   支柱 2                   支柱 3  │
│      ┌─────────┐             ┌─────────┐             ┌─────────┐ │
│      │  探索   │             │  战斗   │             │  成长   │ │
│      └─────────┘             └─────────┘             └─────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 支柱 1：探索 (Exploration)

| 要素 | 描述 |
|------|------|
| **互联世界** | 多区域无缝连接，通过捷径、电梯、传送门串联成一个整体地图 |
| **隐藏内容** | 隐藏墙壁、隐藏房间、隐藏Boss，奖励好奇心 |
| **地图系统** | 探索过的区域自动标记，未探索区域保持迷雾，直观感知探索进度 |
| **环境叙事** | 通过场景废墟、残骸、壁画讲述世界故事 |
| **能力门控** | 不同区域入口需要特定能力才能通过，形成天然的探索顺序 |

### 支柱 2：战斗 (Combat)

| 要素 | 描述 |
|------|------|
| **流畅操作** | 近战/远程攻击即时响应，移动和攻击无缝衔接 |
| **敌人多样性** | 不同敌人有不同的攻击模式和行为，需要不同应对策略 |
| **Boss 战** | 每个区域有独特的 Boss，多阶段战斗，考验玩家对能力的掌握 |
| **打击反馈** | 顿帧、屏幕震动、击退、闪红等多层次反馈 |
| **技能组合** | 获取的技能不仅用于探索，也丰富战斗玩法 |

### 支柱 3：成长 (Growth)

| 要素 | 描述 |
|------|------|
| **能力获取** | 击败 Boss 或找到隐藏道具解锁新能力，改变移动/战斗方式 |
| **数值成长** | 击败敌人获得经验升级，提升 HP、ATK 等属性 |
| **装备收集** | 收集武器、护甲、饰品，自由搭配组合 |
| **碎片收集** | 散布在世界各处的生命碎片、魔力碎片，永久提升上限 |
| **技能解锁** | 能力列表逐步扩充，后期可自由切换和组合 |

## 玩家主要目标与挑战

### 短期目标（单次游戏会话）

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  □ 探索当前区域，填充地图                                     │
│  □ 找到存档点，安全保存进度                                   │
│  □ 击败区域的精英敌人                                        │
│  □ 收集沿途的资源和碎片                                      │
│  □ 找到通往下个区域的路径                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 中期目标（区域级）

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  □ 击败当前区域的 Boss                                       │
│  □ 获得关键能力，解锁之前无法到达的区域                       │
│  □ 发现隐藏区域 / 隐藏 Boss                                  │
│  □ 完成 NPC 的支线任务                                       │
│  □ 提升角色等级，强化装备                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 长期目标（游戏级）

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  □ 击败最终 Boss，完成主线剧情                               │
│  □ 100% 地图探索度                                           │
│  □ 全能力 / 全装备 / 全碎片收集                              │
│  □ 全支线任务完成                                            │
│  □ 全 Boss 击败（含隐藏 Boss）                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 主要挑战

| 挑战类型 | 描述 | 设计意图 |
|----------|------|----------|
| **操作挑战** | 精确平台跳跃，限时通过机关 | 考验玩家操作技巧 |
| **策略挑战** | 不同敌人需要不同应对，Boss 需要找出规律 | 考验玩家观察与思考 |
| **探索挑战** | 隐藏路径需要仔细观察环境线索 | 奖励好奇心 |
| **资源挑战** | HP/MP 有限，需要在存档点之间合理分配 | 制造紧张感 |
| **知识挑战** | 记住之前无法通过的门/路，获取新能力后回溯 | 强化世界记忆 |

## 核心循环阶段

| 阶段 | 玩家行为 | 系统反馈 | 心理驱动 |
|------|----------|----------|----------|
| **1. 探索** | 在未知区域移动，观察环境，触碰边界 | 地图逐步揭露，发现新敌人/新场景 | 好奇心、新鲜感 |
| **2. 遭遇** | 接敌或遇到无法通过的障碍 | 战斗开始 / 看到能力门标记 | 紧张感、挑战欲 |
| **3. 应对** | 战斗（攻击/闪避）或绕过障碍 | 击败敌人获得经验/掉落；遇到死路标记地图 | 掌控感、策略满足 |
| **4. 受阻** | 被强敌击败 / 碰到需要新能力的门 | 死亡返回存档点 / 看到"需要XX能力"提示 | 挫败感 → 求胜欲 |
| **5. 成长** | 探索其他路线 → 击败Boss → 获得新能力 | 能力解锁演出，世界新区域变得可达 | 成就感、力量感 |
| **6. 回溯** | 返回之前无法通过的位置 | 用新能力通过，发现新区域/隐藏内容 | 满足感、"原来如此" |
| **7. 突破** | 探索新区域，推进主线进度 | 世界观信息揭露，剧情推进 | 叙事满足、进度感 |
| **8. 沉淀** | 在存档点休息，整理装备，查看地图 | 存档，地图更新，数据记录 | 安全感、准备感 |

```
│       微循环（30秒）            中循环（15-30分钟）               │

│   │  ↓   │                   │      ↓       │                  │

│   │  ↓   │                   │      ↓       │                  │
│   │ 奖励  │                   │  找到Boss房  │                  │

│                               │  击败Boss    │                  │

│                                      │                          │

│   │  ↓               │        │      ↓       │                  │

```

## 一句话设计宣言

> **"每一次回头，都是一次全新的前进。"**

类银河城的魅力不在于一直向前，而在于获得新能力后回到熟悉的道路，用全新的视角发现全新的可能。
