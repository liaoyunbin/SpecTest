# 时间轴编辑器

## 问题陈述

当前 Unity 项目缺少一个统一的时间轴编辑工具，无法直观地编排和管理基于时间线的动画、音频、事件等内容。需要一个可视化编辑器来：
- 以分层结构（组 → 轨道 → 剪辑块）组织时间线内容
- 支持拖拽排序以灵活调整轨道和分组的位置
- 提供播放控制与时间标尺操作

## 提议方案

开发一个自定义 Unity EditorWindow，提供两部分面板：

**TOP 面板（命令与控制中心）**：包含文件操作（新建/打开/保存）、播放控制（播放/暂停/停止）、时间码显示和帧率设置。

**时间轴面板（核心编辑区）**：
- 左侧轨道头：树形层级展示 Group → Track，支持折叠/展开和拖拽排序
- 右侧剪辑区：时间标尺 + 各轨道的剪辑块渲染，支持播放头拖动和剪辑块拖拽/缩放

数据模型为三层嵌套结构：TimelineData → List\<Group\> → List\<Track\> → List\<Clip\>，不包含关键帧。

## 预期成果

- 可视化的 Group/Track/Clip 三层时间轴编辑器
- 灵活的拖拽排序（Group 全列表可拖，Track 仅同 Group 内可拖）
- 时间标尺缩放和视口平移
- 独立的垂直/水平滚动控制

## 非目标

- 不实现 Keyframe 关键帧编辑
- 不实现运行时播放集成
- 不实现导出功能

## 相关约束

- 基于 Unity EditorWindow 开发
- 项目已安装 `com.unity.timeline` 和 `com.unity.ugui` 包
- 所有产出物使用简体中文
