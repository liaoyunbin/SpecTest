# NPC 系统

## 问题陈述

需要 NPC 交互系统，包括对话、商店和任务触发。

## 提议方案

实现 NPC 交互触发 + 对话树系统。

## 核心功能

- NPC 交互触发 (Trigger)
- 对话树系统 (Luban 配置)
- 商店交互 (买卖物品)
- 任务触发/完成
- 状态持久 (已对话/已交易)

## 预期成果

- NPC 基类
- 对话树引擎
- 商店功能

## 相关约束

- 依赖 ui-manager (对话/商店面板)
- 依赖 config-manager (对话内容、商店物品表)
- 依赖 event-bus (QuestAccepted 事件)
