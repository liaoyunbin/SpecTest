# 规则怪谈，我制定规则 - 设定文档

## 一、世界观设定

### 1.1 核心概念
**规则污染**：概念层面的现实扭曲，"无序"对"有序"的侵蚀
**混乱邪神**："无序"这一概念的人格化，不需要信徒的存在
**现实扭曲**：当规则被污染，现实开始遵循混乱逻辑

### 1.2 灾难背景
**爆发时间**：污染第30天开始，持续约1年半（至第540天）
**影响范围**：全球性规则污染，现实扭曲程度不一
**表现形式**：所有诡异以混乱、矛盾、不符合格式规范的文字规则呈现

### 1.3 规则污染分级体系
**表层污染（轻度）**：规则逻辑基本完整，仅有少量漏洞
**结构污染（中度）**：规则矛盾明显，存在致命陷阱
**本质污染（重度）**：规则完全混乱，现实扭曲严重
**概念污染（终极）**：规则本身成为矛盾的概念

### 1.4 主要地点
**丽景苑小区**：高档住宅小区，被血雾笼罩的秩序孤岛
**1号楼**：林晚的初始根据地，首个安全区建立地
**其他区域**：2号楼、地下车库、公共区域等存在不同规则污染

### 1.5 哲学内涵
**核心矛盾**：秩序与混乱的永恒辩证关系
**深层意义**：规则污染是"无序"试图证明"有序"的局限性
**文明隐喻**：安全区建立是人类文明在混乱中重建的缩影

## 二、人物设定

### 2.1 主角 - 林晚
**基本信息**：
- 姓名：林晚（28岁，沿用原主姓名）
- 身份：穿越者（来自21世纪的重度强迫症患者）
- 职业：前资深物业管理员
- 性格：重度规则强迫症 + 极致理性

**强迫症特质**：
- **视觉强迫**：无法忍受格式错误，瞳孔会生理性收缩
- **逻辑强迫**：对矛盾规则产生头痛等生理不适
- **秩序强迫**：一切必须按规则运行，不可有例外
- **文本强迫**：字体大小、标点符号、段落格式必须规范统一

**行为模式**：
- 永远从根源解决问题
- 理性决策，不考虑人情
- 对混乱的生理不适真实强烈
- 随身携带红色批注笔

### 2.2 核心团队成员
**张叔**：水电工程师，技术骨干，沉稳可靠
**李四**：前保安队长，战斗经验丰富，执行力强
**刘姐**：社区医生，医疗知识丰富，细心体贴
**小王**：年轻程序员，系统操作熟练，思维活跃
**老陈**：退休教师，知识渊博，善于协调

### 2.3 反派角色
**诡异教徒**：信奉原始混乱规则的狂热分子
**规则寄生体**：能主动散布混乱规则的污染源
**混乱邪神**："无序"的人格化，终极对手

### 2.4 特殊角色
**会哭的雕像**：情绪宣泄异常的实体，需要规范化管理
**无限循环楼梯**：空间扭曲现象，可改造为实用设施
**死亡广播**：强制回应规则，需要标准化改造

## 三、能力设定

### 3.1 核心能力 - 规则纠错
**能力本质**：看穿规则漏洞，通过修正改写规则效力
**表现形式**：用红色批注笔在规则文本上进行批注修订

**具体功能**：
- **看穿漏洞**：能直观看到诡异规则中的逻辑漏洞和矛盾之处
- **规则修正**：通过批注、修订规则文本，改写规则效力
- **规则转化**：将"绝对杀人规则"弱化为"限制性规则"或"可规避流程"
- **规则固化**：修正后的规则会固化生效，形成稳定的秩序领域

### 3.2 规则解析系统
**系统来源**：来自已战胜"无序"的高维文明遗产
**功能模块**：
- **规则可视化**：将抽象规则转化为可分析结构
- **漏洞标记**：识别规则中的逻辑漏洞和矛盾点
- **修正模拟**：预测修正效果和风险评估
- **能量管理**：分析消耗精神力，需要合理使用

**使用限制**：
- 只能分析已发现的规则
- 能量消耗随分析复杂度增加
- 升级需要积累修正经验

### 3.3 神器设定
**红色批注笔**：
- 无法被污染，墨水无限
- 修正效果永久固化
- 随着使用解锁新功能
- 是"有序"力量的具象化

**已收容异常规则目录**：
- 自动归档所有修正后的规则
- 提供规则查询和参考功能
- 记录规则修正的历史和经验
- 是秩序建立的见证和档案

### 3.4 能力成长体系
**初级阶段**：单个规则修正，安全区建立
**中级阶段**：系统化规则管理，异常收容
**高级阶段**：规则定义，秩序领域扩张
**终极阶段**：概念层面规则制定，对抗混乱源头
