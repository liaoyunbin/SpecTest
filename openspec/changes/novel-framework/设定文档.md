# 规则怪谈，我制定规则 - 设定文档

## 一、世界观设定

### 1.1 核心概念
**规则污染**：世界被诡异规则扭曲，变得危险又混乱
**混乱邪神**：代表绝对混乱的终极BOSS
**现实扭曲**：被污染的地方会变得不正常

### 1.2 灾难背景
**爆发时间**：污染第30天开始，持续约1年半（至第540天）
**影响范围**：全球性规则污染，各地危险程度不同
**表现形式**：所有诡异都以混乱、矛盾的文字规则出现

### 1.3 混乱程度等级
**轻度污染**：规则基本正常，只有少量小问题
**中度污染**：规则有明显矛盾，存在致命陷阱
**重度污染**：规则完全混乱，现实扭曲严重
**终极污染**：规则本身就充满矛盾，极度危险

### 1.4 主要地点
**丽景苑小区**：高档住宅小区，被血雾笼罩的安全岛
**1号楼**：林晚的初始根据地，第一个安全区
**其他区域**：2号楼、地下车库、公共区域等有不同程度的危险

### 1.5 故事背景
**核心矛盾**：主角用规则对抗混乱，保护幸存者
**故事主线**：建立安全区，对抗敌人，重建文明

## 二、人物设定

### 2.1 主角 - 林晚
**基本信息**：
- 姓名：林晚（28岁）
- 身份：穿越者（来自21世纪的重度强迫症患者）
- 职业：前资深物业管理员
- 性格：冷静理智，外冷内热

**强迫症特质**：
- **视觉强迫**：看到格式错误会浑身难受，瞳孔收缩
- **逻辑强迫**：遇到矛盾规则会头痛，必须马上修正
- **秩序强迫**：一切必须按规则运行，容不得半点瑕疵
- **文本强迫**：字体大小、标点符号必须规范统一

**行为模式**：
- 永远从根源解决问题
- 冷静分析，果断执行
- 随身携带标志性的红色批注笔

### 2.2 核心团队成员
**张叔**：水电工程师，技术厉害，沉稳可靠
**李四**：前保安队长，身手好，执行力强
**刘姐**：社区医生，医术高明，细心体贴
**小王**：年轻程序员，电脑技术好，思维活跃
**老陈**：退休教师，知识渊博，善于协调

### 2.3 反派角色
**诡异教徒**：信奉混乱的疯子，到处搞破坏
**规则寄生体**：主动传播污染的怪物
**混乱邪神**：最终BOSS，代表绝对的混乱

### 2.4 特殊角色
**会哭的雕像**：会哭的异常实体，需要好好管理
**无限循环楼梯**：能让人无限循环的楼梯，可改造为健身设施
**死亡广播**：必须回应的恐怖广播，需要改造

## 三、能力设定

### 3.1 核心能力 - 规则纠错
**能力本质**：看穿规则漏洞，通过修正改写规则效力
**表现形式**：用红色批注笔在规则文本上进行批注修订

**具体功能**：
- **看穿漏洞**：能看到诡异规则中的逻辑漏洞和矛盾
- **规则修正**：通过批注改写规则，让它变得安全
- **规则转化**：将致命规则变成可规避的流程
- **规则固化**：修正后的规则会固定生效，形成安全区域

### 3.2 规则解析系统
**系统来源**：来自高维文明的神秘遗产
**功能模块**：
- **规则可视化**：把抽象规则变成能看到的结构
- **漏洞标记**：找出规则中的问题和矛盾
- **修正模拟**：预测修正后的效果
- **能量管理**：分析消耗的精神力

**使用限制**：
- 只能分析已经发现的规则
- 规则越复杂，消耗的能量越多
- 需要积累经验才能升级

### 3.3 神器设定
**红色批注笔**：
- 不会被污染，墨水用不完
- 修正效果永久有效
- 用得越多，解锁的功能越强
- 是秩序力量的象征

**已收容异常规则目录**：
- 自动保存所有修正后的规则
- 可以查询和参考以前的规则
- 记录规则修正的历史
- 是秩序建立的见证

### 3.4 能力成长体系
**初级阶段**：单个规则修正，安全区建立
**中级阶段**：系统化规则管理，异常收容
**高级阶段**：规则定义，秩序领域扩张
**终极阶段**：概念层面规则制定，对抗混乱源头
