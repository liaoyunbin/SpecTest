# EventBus 事件系统

## 问题陈述

系统间通信缺乏统一机制，直接引用导致耦合严重。需要一个全局事件系统实现发布/订阅解耦。

## 提议方案

实现轻量级 EventBus 单例，支持全局事件发布/订阅，用于所有跨系统通信。

## 核心功能

- 全局事件发布 (`Publish`)
- 任意系统订阅 (`Subscribe`)
- 取消订阅 (`Unsubscribe`)
- Debug 模式事件日志

## 预期成果

- 系统间解耦通信
- 14 个预定义全局事件
- 零额外依赖

## 相关约束

- 纯 C# 单例，不依赖 MonoBehaviour
- 事件名使用英文过去式命名
